## 如何使用：

从Release下载压缩包并解压

自行下载gpuz软件，将exe文件放至解压后文件夹的根目录(与start.bat文件同级)

运行start.bat

打开VRChat，确保OSC已打开。硬件信息会自动发送至聊天框

## How to use:

Download and extract the package from Release.

Download the gpuz software and place the .exe file in the root directory of the extracted folder (at the same level as the start.bat file).

Run start.bat.

Open VRChat and ensure OSC is enabled. Hardware information will be automatically sent to the chat window.
